# Flower Image Classification using CNN

## Overview

This project demonstrates a Convolutional Neural Network (CNN) for image classification using TensorFlow and Keras. The model is trained on a flower image dataset and uses image preprocessing and augmentation techniques to improve learning performance.

## Features

* Image augmentation using ImageDataGenerator
* Convolutional Neural Network (CNN) architecture
* Image resizing and normalization
* Training and validation performance visualization
* Model saving for future inference

## Technologies Used

* Python
* TensorFlow
* Keras
* NumPy
* Matplotlib
* PIL

## Model Architecture

* Conv2D (32 filters)
* MaxPooling2D
* Conv2D (64 filters)
* MaxPooling2D
* Conv2D (128 filters)
* MaxPooling2D
* Flatten Layer
* Dense Layer (512 neurons)
* Softmax Output Layer

## Results

The model achieved approximately 81% accuracy during training and validation.

## Learning Outcomes

* Model training and evaluation
* Performance visualization using loss and accuracy curves
